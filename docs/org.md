# Multi Timer

## 概要

複数のタイマーを設定できるウェブアプリ。
Google のタイマーの複数版。

## 機能

- タイマーを複数設定できる
- タイマーには以下のボタンがある
  - スタート / ストップ: スタートマークのボタンをクリックするとタイマーのカウントダウンが始まりスットプマークに変わり、ストップマークのボタンをクリックするとタイマーのカウントダウンは停止してスタートマークに変わる
  - リセット: タイマーを初期値に戻す
  - 最大化: 一つのタイマーを画面いっぱいに表示する
  - 削除: タイマーを削除する
  - 編集: タイマーの設定を変更する
  - 「+ 0:10」: タイマーの残り時間を 10 秒増やす
  - 「+ 1:00」: タイマーの残り時間を 1 分増やす
- タイマーは 1 秒単位で設定、表示する
- タイマーの最大設定時間は 99:59:59 とする
- 円の中心に残り時間を表示し、円の縁はプログレスバーのように進行度合いによって色をつける

## 確定仕様

設計レビューで確定した詳細仕様。

### タイマーの基本

| 項目 | 内容 |
| --- | --- |
| 初期状態 | 1 個のタイマーを表示する（初期時間は 1:00） |
| 追加 | 「+」ボタンでタイマーを追加する |
| 表示形式 | Google タイマー風（1 時間未満は `MM:SS`、それ以上は `H:MM:SS` など） |
| 設定単位 | 1 秒単位 |
| 上限 | 99:59:59 |
| 永続化 | なし（リロードでリセット） |
| レイアウト | レスポンシブグリッド |

### ボタン挙動

| ボタン | 挙動 |
| --- | --- |
| スタート / ストップ | カウントダウンを開始・一時停止し、アイコンを切り替える |
| リセット | 初期設定時間に戻し、停止状態にする |
| 最大化 | 1 つのタイマーを画面いっぱいに表示する。一覧に戻るのは最小化ボタンと Esc キーの両方 |
| 削除 | 確認なしで即削除する |
| 編集 | Google タイマー風 UI で設定を変更する。実行中は不可 |
| + 0:10 | 10 秒加算する。停止中のみ押せる |
| + 1:00 | 1 分加算する。停止中のみ押せる |

### 制約・完了時

- 残り時間が 0:00 のときはスタートできない
- 「+ 0:10」「+ 1:00」で 99:59:59 を超える場合は加算せず、99:59:59 が上限であることを表示する
- カウントダウンが 0:00 になったときは、アラーム音は鳴らさない
- 完了時はタイマーの色が変わり、ブラウザタブの favicon で完了を示す（いずれか 1 つでも完了したら反映。すべての完了が解除されたら通常に戻す。）
- 完了状態は任意の操作（リセット、編集など。スタートは除く。）で解除する
- 複数タイマーは同時に実行できる
- 設定されているタイマーが 1 個のとき、タイマーを削除することができない

### UI / デザイン

- シンプルで洗練された見た目とする
- ライトモードとダークモードの両方に対応し、アプリ内トグルで切り替える
- 円形プログレスは 12 時方向から反時計回りに減少する

### 実装時のデフォルト

以下は未指定のため、実装時に合理的なデフォルトを採用する。

| 項目 | デフォルト |
| --- | --- |
| 完了時の色 | アクセントカラー（例: 緑やオレンジ）でリングを変更する |
| 完了 favicon | 通常 favicon にチェックマーク等を重ねた別アイコン |
| 上限超過の表示 | インラインの短いメッセージ |
| タイマーの名前 / ラベル | なし（時間のみ表示） |
